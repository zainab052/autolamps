POS EXTRA UTILITIES

Feature 1: Restrict user to view his default POS only

This feature restrict the current logged-in user to view only his default Point Of Sales, not all.

Feature 2: Prevent Order Validation If Customer Is Not Selected

This feature raises an alert if user tries to validate an order without selecting a customer.

Feature 3,4 and 5: Disabling Default Features

These features allows the user to disable the functionality of the Price, Discount, and Delete button in the POS.

Disable Price Modification

This feature disables the functionality of price button in POS. It prevents any modifications in price.

Disable Discounts

This feature disables the functionality of Discount button in POS.

Disable Delete

This feature disables the functionality of Delete button in POS. However, delete works normally for decreasing quantity of the products in orderlines.

Feature 6: Allow Only Increase In Price

This features modifies the default behaviour of Price button. It prevents the user from decreasing the price of the products in orderlines. Only price increase is permitted.
