POS Warehouse Management

POS Warehouse Management is a module to manage multiple warehouses/locations within POS session. It shows and manage products stock in POS session. By default in POS only one stock location can be selected at one time. This module allows the POS user to select more than one stock locations in POS, So that if any product is out of stock in POS Session then POS user can order product from other stock locations.

Features
  • Display the current stock of products in running session.
  • Seller can select multiple stock locations for a single Point Of Sale by setting the other related stock locations
  • If product is out-of-stock in main stock location, seller can add product from other related stock locations.
  • Stock will be updated automatically after order validation
  • POS Session View
  • Backend Configuration

Create Related Pickings in Ready/Done state

Related pickings are created if a product is ordered from other related stock locations. Check 'Create Related Pickings in Ready State ' option if you wish to create related pickings in the Ready state so that you can validate these related pickings in the backend. If 'Create Related Pickings in Ready State' option is unchecked then related pickings will be validated automatically and will be in the done state.
